### Ollama setup
1. Install Ollama using Homebrew:
   ```bash
   brew install ollama
   ```
2. Verify the installation:
   ```bash
   ollama version
   ```
3. Download the desired model (e.g., `gpt-oss:20b`):
   ```bash
   ollama pull gpt-oss:20b
   ```
